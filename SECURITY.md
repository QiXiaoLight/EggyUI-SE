# EggyUI-SE 安全策略

## 项目定位与安全承诺

EggyUI-SE 是一个个人/小团队维护的开源美化项目，专注于以不修改系统核心文件的方式提供蛋仔派对风格的 Windows 桌面美化。我们深知，作为一个小型项目，资源有限，但我们对用户安全的重视从未降低。

## 安全原则

1. **不修改系统核心文件**：EggyUI-SE 的所有美化功能均不替换或修改 Windows 系统关键文件（如 `imageres.dll.mun`、系统内核文件等），从根本上规避系统崩溃、蓝屏等风险。

2. **无捆绑、无后门**：项目安装包不含任何捆绑软件、广告插件、后门程序或数据收集代码。所有功能均为透明实现。

3. **代码开源**：项目核心代码已在 GitHub 公开，任何开发者均可审查。我们欢迎社区共同监督代码安全性。

## 报告安全漏洞

如果您发现 EggyUI-SE 存在安全漏洞（如代码注入、权限提升、不当数据处理等），请通过以下渠道联系我们，**请勿公开披露**，以便我们有时间修复问题：

- **GitHub Issues**：在本仓库提交私密 Issue（勾选“只有维护者可见”）

## 已知风险与用户须知

作为个人/小团队项目，我们坦诚以下几点：

- **测试覆盖有限**：项目主要在 Windows 11 主流版本上测试，老版本系统可能存在未覆盖的兼容性问题。建议在虚拟机或备用设备上先行体验。
- **依赖第三方工具**：部分功能可能依赖系统自带工具或开源库，我们已尽量选用稳定版本，但无法完全排除第三方依赖的潜在风险。
- **用户责任**：请从官方渠道（GitHub 仓库、官方群公告）获取安装包，避免使用来路不明的修改版。因使用非官方版本导致的问题，项目维护者不承担责任。
